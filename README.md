# Getting Started with Go Programming: A Beginner-Friendly Toolkit Using Generative AI

## 📌 Project Description
This project is a beginner-friendly motivational command-line application built using the Go programming language. It demonstrates how to use Generative AI to learn a new technology, set up a development environment, and build a simple interactive program.

The application prompts the user to enter their name and displays a personalized greeting along with an encouraging motivational message.

---

## 🎯 Project Objective
- Learn the basics of Go (Golang)
- Build a simple interactive program
- Use AI tools to guide learning and problem-solving
- Document the entire process for beginners

---

## 🛠️ Technologies Used
- Go (Golang)
- Command Line Interface (CLI)
- Generative AI (ai.moringaschool.com)

---

## 💻 System Requirements
- Windows OS (or Mac/Linux)
- Go installed on your machine
- A code editor (VS Code or Notepad)

---

## ⚙️ Installation & Setup

### 1. Install Go
Download Go from the official website:
https://go.dev/dl/

Run the installer and follow the setup instructions.

---

### 2. Verify Installation
Open Command Prompt and run:

go version

You should see output similar to:
go version go1.xx windows/amd64

---

## ▶️ How to Run the Project

1. Open terminal or Command Prompt
2. Navigate to the project folder:
   cd go-project

3. Run the program:
   go run main.go

## ⚠️ Common Issues & Fixes

### Issue 1:
'go' is not recognized as an internal or external command

Fix:
Install Go and ensure it is added to system PATH.

---

### Issue 2:
expected 'package', found 'EOF'

Fix:
Ensure the Go file contains valid code and is properly saved.
